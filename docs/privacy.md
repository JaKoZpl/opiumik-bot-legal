# Политика конфиденциальности (Privacy Policy) — Opiumik Bot

Дата вступления в силу: 2026-01-04

Эта политика описывает, какие данные может обрабатывать Discord-бот **Opiumik** (“Бот”) и как они используются.

## 1. Какие данные обрабатываются
Бот может сохранять следующие данные (в зависимости от используемых функций):
- Discord ID пользователя (userId),
- Discord username / tag (например, user.tag),
- данные из формы заявки (например: nickname/static, OOC-информация, цель, опыт),
- служебные ID Discord (ID каналов/сообщений/гильдии) для работы тикетов/логов,
- данные по контракт-форме (выбранный контракт, список выбранных пользователей).

Бот **не** запрашивает и не хранит пароли, токены пользователей или платежные данные.

## 2. Зачем нужны эти данные
Данные используются для:
- обработки заявок на вступление (принято/отклонено/на рассмотрении),
- ведения истории заявок и логов действий,
- публикации и учета контрактов,
- работы модерационных команд на сервере.

## 3. Где и как хранятся данные
Данные могут храниться в базе данных (например, MongoDB) на сервере, где запущен Бот, и используются только для функций Бота.

## 4. Срок хранения
Данные хранятся столько, сколько необходимо для работы функций:
- заявки/контракты/логи могут храниться длительно для истории и админ-контроля,
- по запросу пользователя данные могут быть удалены (см. раздел 6).

## 5. Передача данных третьим лицам
Бот не продаёт и не передает данные третьим лицам.
Данные могут быть обработаны платформой Discord согласно их политике.

## 6. Права пользователя и удаление данных
Вы можете запросить:
- просмотр данных, связанных с вашим Discord ID,
- удаление ваших данных.
